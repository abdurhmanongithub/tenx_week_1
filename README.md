# **Task 1**
This repository contains the analysis of a solar dataset as part of Task 1. The focus is on performing Exploratory Data Analysis (EDA) to extract insights, visualize patterns, and clean the data for further use.
---
## **Steps Completed**

1. **Summary Statistics**:
   - Calculated metrics like mean, median, and standard deviation for numerical columns.

2. **Data Quality Check**:
   - Identified missing values and outliers in key columns such as `GHI`, `DNI`, `DHI`, `ModA`, `ModB`, and `WS`.

3. **Visualizations**:
   - Time series plots for variables like `GHI`, `DNI`, `DHI`, and `Tamb`.
   - Histograms for visualizing the data distribution.
   - Correlation heatmaps to understand relationships between solar radiation, temperature, and wind conditions.

4. **Data Cleaning**:
   - Handled missing values, removed anomalies, and prepared a cleaned dataset.
