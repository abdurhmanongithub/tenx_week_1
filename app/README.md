## Solar Dataset Dashboard

This project includes a Streamlit dashboard to explore and visualize solar data interactively.

### Features
- View raw solar dataset.
- Visualize trends in solar irradiance (GHI, DNI, DHI) and temperature over time.
- Apply filters (e.g., date range) to refine analysis.
